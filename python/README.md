# Python Katas

- [Arithmetic](https://www.codewars.com/kata/583f158ea20cfcbeb400000a)

## To setup locally
```
pip install -r requirements.txt
```

## Testing

### Run tests
```
pytest
```

### Show test name when running tests
```
pytest -rA
```

### To run a report and watch tests
```
make watch-test-report
```
